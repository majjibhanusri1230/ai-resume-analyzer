# 💼 AI Resume Analyzer
🚀 A smart web app that analyzes resumes and gives skill suggestions, score, and improvements.
## 🔥 Features
- 📄 Upload Resume (PDF)
- 🧠 Extract Skills Automatically
- ⭐ Resume Score (out of 100)
- 📊 Skill Progress Bar
- 🎯 Job Role Matching
- 💡 Smart Suggestions
## 🛠 Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Python (Flask)
- NLP: SpaCy
