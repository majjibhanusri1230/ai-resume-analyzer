# 💼 AI Resume Analyzer

🚀 A smart web application that analyzes resumes using AI/NLP and provides skill insights, suggestions, scoring, and job role matching.

---

## 🔥 Features
- 📄 Upload Resume (PDF)
- 🧠 Extract Skills Automatically using NLP
- ⭐ Resume Score (out of 100)
- 📊 Skill Progress Bar Visualization
- 🎯 Job Role Matching (Frontend, Backend, Full Stack)
- 💡 Smart Suggestions for Improvement
- 💬 Confidence Boosting Feedback

---

## 🛠 Tech Stack
- Frontend: HTML, CSS, JavaScript  
- Backend: Python (Flask)  
- NLP: SpaCy  
- Libraries: pdfplumber, flask-cors  

---

## ▶️ How to Run Locally

1. Clone the repository
git clone https://github.com/majjibhanusri1230/ai-resume-analyzer.git
2. Go to backend folder
cd backend
3. Create virtual environment
python -m venv venv
4. Activate environment
venv\Scripts\activate
5. Install dependencies
pip install -r requirements.txt
6. Run Flask server
python app.py
7. Open frontend/index.html in browser
---
## 📊 How It Works
1. User uploads resume (PDF)
2. Backend extracts text using pdfplumber
3. NLP processes and identifies skills
4. System:
   - Calculates score
   - Matches job roles
   - Gives suggestions
5. Frontend displays results with UI
